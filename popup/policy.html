Політика конфіденційності для OTRSPost

Дата набуття чинності: 25 липня 2025 р.

1. Яку інформацію ми збираємо

Розширення OTRSPost збирає наступні типи даних:

  • Дані про тікети з OTRS: Номер, заголовок, та текст заявки для відправки сповіщень в Telegram.
  • Автентифікаційні дані: Ваш логін та пароль для автоматичного входу в систему OTRS та для отримання токену доступу до налаштувань, що зберігаються в базі даних Firebase.
  • Налаштування розширення: Інтервал перевірки нових тікетів.

2. Як ми використовуємо вашу інформацію

Зібрані дані використовуються виключно для забезпечення основної функціональності розширення:

  • Для моніторингу нових тікетів у системі OTRS.
  • Для надсилання сповіщень про нові тікети у вказаний вами Telegram-чат.
  • Для автоматичної авторизації в OTRS, щоб підтримувати активну сесію.
  • Для збереження та синхронізації ваших налаштувань.

3. Передача даних третім сторонам

Ми передаємо ваші дані наступним сервісам:

  • Telegram API: Для надсилання сповіщень.
  • Google Firebase: Для зберігання налаштувань розширення та автентифікації.
Ми не продаємо та не передаємо ваші особисті дані іншим третім сторонам.

4. Зберігання та безпека даних

Ваші автентифікаційні дані (логін та пароль) та налаштування зберігаються локально на вашому комп'ютері за допомогою API сховища браузера (`browser.storage.local`). Вони використовуються для взаємодії з OTRS та Firebase, але не зберігаються на наших серверах.

5. Ваші права та контроль

Ви маєте повний контроль над своїми даними. Ви можете видалити свої дані, видаливши розширення з браузера, або очистивши дані у налаштуваннях розширення.

6. Зміни в політиці конфіденційності

Ми можемо оновлювати цю політику конфіденційності. Про будь-які зміни ми будемо повідомляти через оновлення розширення.
